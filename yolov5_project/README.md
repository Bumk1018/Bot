# YOLOv5 Object Detection Project

이 프로젝트는 [Ultralytics YOLOv5](https://github.com/ultralytics/yolov5)를 사용하여 test_images 폴더에 있는 이미지에서 객체를 탐지하는 예제입니다.

## 🔧 설치 방법
```bash
git clone --recurse-submodules https://github.com/your-username/yolov5_project.git
cd yolov5_project
pip install -r yolov5/requirements.txt
```

## 📦 모델 다운로드 (자동)
코드를 실행하면 `yolov5s.pt`가 자동으로 다운로드됩니다.

## 🚀 실행 방법
```bash
python run_random_detect.py
```
